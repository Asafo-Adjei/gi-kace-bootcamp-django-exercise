#NSS Marketplace (Django)

A simple marketplace web application built with Django where users can register, log in, and view listed products. Admin users can manage and control item visibility.

---

##Features

- Custom User Model
- User Registration, Login & Logout
- Admin Panel
- Add Marketplace Items (Admin)
- Active / Inactive Item Control
- Product Image Upload
- Styled UI with CSS
- Item Detail View Page
- Secure Secret Key using `.env`
- Basic Unit Tests
